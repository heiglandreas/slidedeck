### JUnitDiff

* What changed between two test-runs



### JUnitDiff

```bash
    git checkout master
    phpunit --log-junit master.xml
    git checkout feature/branch
    phpunit --log-junit feature.xml
    junitdiff compare master.xml feature.xml
```
